# DevTrack 优化方案

## 参考项目分析（game-dev-tracker）

参考项目亮点功能：
- 5 页面导航：概览 → 项目 → 开发日志 → 数据分析 → 任务看板
- 项目/工作区概念，支持多项目切换
- 里程碑进度指示器（0/13）
- 成就系统
- 任务多视图切换（看板/列表/专注）
- 搜索（Ctrl+K 快捷键）
- 日/夜模式切换
- 中英双语
- 数据分析页（图表）

## 本次优化内容

### 问题修复
1. ✅ 一键启动脚本 `start.bat`
2. ✅ npm 路径问题修复

### 架构升级：项目/工作区系统
1. **新增「项目」实体** - 每个项目有独立的仪表盘、任务、时间线、日记
2. **新增项目选择/管理页面** - 创建、切换、删除项目
3. **路由重构** - `/:projectId/dashboard`, `/:projectId/tasks`, `/:projectId/timeline`, `/:projectId/diary`, `/:projectId/analytics`

### 里程碑重设计
- 里程碑属于项目，由开发者自定义名称
- 可以关联任务
- 自动计算完成进度
- 里程碑页面展示进度环/条

### 新增数据分析页（图表重点！）
- 任务完成趋势折线图
- 任务状态分布饼图
- 每日提交/活动热力图
- 优先级分布柱状图
- 使用 Chart.js + react-chartjs-2

### 新增功能
- 任务列表视图（看板/列表切换）
- 搜索/命令面板（Ctrl+K）
- 成就系统（自动解锁）
- 设置页面（主题切换、数据导出/导入）
- 公网可访问标题栏（workspace bar）
